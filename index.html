Research & Projects

Selected ongoing and completed research projects.

AI-generated Text & Code Detection Ongoing

Deep learning detectors to distinguish LLM-generated and human-written text and source code using CNN, GRU, hybrid LSTM–CNN, and transformer architectures (e.g., DistilBERT).

  • Evaluation: accuracy, precision, recall, F1-score on diverse text/code datasets
  • Applications: AI safety, authenticity verification, academic integrity

Code Smell Detection in Python Ongoing

ML-based classifiers for detecting Large Class and Long Method code smells using XGBoost, LightGBM, AdaBoost, with class-balancing strategies (SMOTE, SMOTEENN).

  • Features: structural, complexity, and size-related metrics
  • Goal: integration with modern static analysis / refactoring workflows

Autism Spectrum Disorder Detection

ML-based early prediction of ASD using behavioral and social communication indicators (AQ1–AQ10).

  • Tested 271 classifiers; top models achieved 97.2%–99.1% accuracy
  • Handled heterogeneity, imbalance, and feature sparsity through preprocessing

Round Robin Scheduling Algorithm Analysis

Analytical and experimental studies of Round Robin scheduling focusing on time quantum optimization.

  • Compared response time, waiting time, turnaround time under varying loads
  • Proposed improvements and performance trade-offs for fair CPU allocation
